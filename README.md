# Wikimint Tools Dataset

This is the official public dataset behind  
**https://www.wikimint.com/tools**

It contains structured, versioned JSON files of free, open-source,
and freemium tools across business, marketing, finance, AI, and SaaS.

## Purpose
This repository exists for:

- Public transparency
- Dataset verification
- Historical versioning
- Record validation
- Research & educational reuse

## How Wikimint Uses This Data
This repo is automatically consumed by the Wikimint website tools directory during build time to generate category pages.

No runtime API calls. Fully static.

## Update Policy
- Every tool addition is a commit
- Every category change is versioned
- All changes are permanent & auditable
